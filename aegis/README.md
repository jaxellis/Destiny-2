# Aegis DIM wishlist generator

This folder was entirely generated by GPT 5.6 Terra High. Accuracy may be missing!

`generate_dim_wishlist.py` reads the local Aegis workbook and emits a DIM
wishlist using the local Bungie manifest index.

Run it from this directory:

```powershell
python generate_dim_wishlist.py
```

The first run downloads the inventory-item definitions linked by
`manifest.json` into `.manifest-cache`; later runs use that cache. It writes
`aegis_strict_dim_wishlist.txt` by default.

The active section contains only S- and A-tier weapons (including future `A+`
or `A-` labels). B through F are emitted as negative DIM item hashes, which
marks every roll on those weapons as trash.

For active weapons, the script creates every strict combination requiring two
recommended plugs in both `Perk 1` and `Perk 2`. A one-option perk cell is the
sole exception and requires that one option. Barrel, magazine, and origin
trait recommendations are emitted in optional variants, matching the behavior
of the Charles Aegis selector without allowing their absence to invalidate a
core roll.

Useful options:

```powershell
# Validate one current worksheet quickly
python generate_dim_wishlist.py --sheet Autos --output autos.txt

# Use a different strictness for the core perk columns
python generate_dim_wishlist.py --required-perks-per-column 3
```
